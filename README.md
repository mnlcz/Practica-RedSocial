# Red Social
## Entrega 1
En la entrega 1 se implementa el módulo persona, respetando la métodologia brindada por el profesor (paradigma y notación). <br>
Además de las funciones dadas por el profesor, se utilizan una serie de funciones extra para facilitar el trabajo. <br>
### Amigos
Para la implementación de los amigos de una persona, se optó por la utilización de vectores y no de listas enlazadas, por lo tanto, el modulo de ConjuntoPersonas.h, no será utilizado.
